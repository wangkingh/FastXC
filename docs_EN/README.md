# FastXC English Documentation

This directory keeps the English project documentation. The repository root
README stays focused on installation, quick-start usage, and the fields most
users edit first.

- [Main README](../README.en.md): installation, quick start, examples, and common fields.
- [Configuration](CONFIGURATION.md): INI fields, SAC path pattern rules, and common values.
- [Outputs](OUTPUTS.md): stage-by-stage workspace outputs and what each artifact means.
- [Tools](TOOLS.md): stage aliases, manual export, conversion, SourcePack repair, StepPack inspection, and plotting tools.
- [Architecture](ARCHITECTURE.md): current data flow and module boundaries.
- [Results](RESULTS.md): public policy for generated result artifacts.
- [Changelog](../CHANGELOG.md): release-facing changes and compatibility notes.
- [中文文档](../docs_ZH/README.md): Chinese documentation index.

Detailed development notes are kept under [changelog/](../changelog/).
